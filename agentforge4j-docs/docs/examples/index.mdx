# Examples

A gallery of runnable AgentForge4j examples. Every sample shown here is compiled and built
from the framework examples sources and included into the docs, never copy-pasted — so a
sample that no longer compiles fails the build.

:::note Work in progress
The gallery entries and their included sources are wired in with the snippet-include
mechanism in a later phase. The examples themselves live in the framework examples module.
:::

# Examples

Every example is a real, runnable module under `agentforge4j-examples/`. The workflow definition
shown for each is **included from source** and validated against the published schema at build time —
never copy-pasted — so an example that drifts from its schema or stops compiling fails the build. Each
example runs deterministically offline against the test provider (no network, no API key).

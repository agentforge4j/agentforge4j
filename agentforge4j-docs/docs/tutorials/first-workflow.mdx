# Your first workflow

A learning-oriented walkthrough that takes you from nothing to a complete, running
workflow.

:::note Work in progress
The end-to-end runnable sources for this tutorial come from the framework examples and are
included via the snippet-include mechanism. This page currently outlines the journey; see
[Examples](../examples/index.mdx) for the compiled sources.
:::

By the end you will have:

- bootstrapped the runtime,
- configured an LLM provider,
- supplied a workflow definition validated against the [schema](../reference/schemas.mdx),
- executed the workflow and observed its [events](../reference/events.mdx) and final
  [status](../reference/statuses.mdx).

When you are comfortable with the basics, move on to the goal-oriented
[How-to Guides](../how-to/configure-llm-provider.mdx) and the [Concepts](../concepts/workflows.mdx).

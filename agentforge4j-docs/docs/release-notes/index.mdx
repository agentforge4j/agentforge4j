# Release notes

This page documents the unreleased development line (`next`). Changes here have not shipped
yet — track them as they land in the framework's
[CHANGELOG.md](https://github.com/agentforge4j/agentforge4j/blob/main/CHANGELOG.md), under its
`[Unreleased]` section.

For the notes of an actual shipped release, use the version selector above to switch to a
released documentation version, or see the
[framework-v0.1.0 GitHub release](https://github.com/agentforge4j/agentforge4j/releases/tag/framework-v0.1.0)
directly. The shipped workflow catalog and the workflow builder release independently, on
their own tracks — see [Releases](https://agentforge4j.org/releases) for all three.
